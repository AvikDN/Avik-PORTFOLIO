
# Avik Portfolio

A sleek, sci-fi themed personal portfolio website built with **HTML, CSS, and JavaScript**.  
Features a sliding side menu, animated navigation links, and a cinematic hero layout.

## ✨ Features
- Responsive layout  
- Slide-out side menu  
- Scramble text hover effects  
- Modern UI / cyberpunk-inspired design  
- Image-based hero section with play button

## 🛠 Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript  
- Google Fonts (Bricolage Grotesque)

## 🚀 Getting Started
1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/Avik-portfolio.git
````

2. Open `index.html` in your browser.
